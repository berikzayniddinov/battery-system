üéØ System Overview

This system monitors battery health and predicts Remaining Useful Life (RUL) using machine learning. It provides real-time monitoring of battery parameters and forecasts when maintenance or replacement is needed.

üîÑ Control Buttons

Update Prediction

Triggers a new RUL prediction using the latest battery data. The model analyzes historical patterns to forecast remaining cycles until battery capacity drops below 80% of original capacity.

Add Sample Data

Generates and adds realistic battery data with simulated degradation. Useful for testing and demonstration without requiring physical battery measurements.

Retrain Model

Retrains the machine learning model with all available historical data. Improves prediction accuracy as more data becomes available over time.

üìä Charts & Visualizations

Capacity Degradation

Shows battery capacity reduction over charge cycles. Steeper slopes indicate faster degradation. Normal batteries show gradual, linear capacity loss.

Voltage & Temperature

Displays operational parameters during charge/discharge cycles. Abnormal voltage drops or temperature spikes may indicate battery health issues or safety concerns.

üìù Data Input Form

The form allows manual input of battery parameters for monitoring. All fields are required for accurate predictions:

  • Battery ID: Unique identifier for the battery unit
  • Voltage: Current voltage level in Volts (V)
  • Current: Electrical current in Amperes (A)
  • Temperature: Operating temperature in Celsius (¬∞C)
  • Capacity: Current capacity in Ampere-hours (Ah)
  • Cycle Number: Total charge-discharge cycle count

üîç Understanding Predictions

RUL (Remaining Useful Life)

Estimated number of cycles until battery capacity drops below 80% of original capacity. Lower values indicate the battery is approaching end-of-life.

Confidence Level

Model's certainty in the prediction (0-100%). Higher values indicate more reliable forecasts based on data quality and pattern consistency.

Current Cycle

Total number of charge-discharge cycles the battery has undergone. Each cycle contributes to gradual capacity degradation.

Machine Learning Model

Uses Random Forest Regressor trained on historical battery data. The model considers capacity degradation trends, voltage characteristics, and temperature patterns.

Data Processing

Automatically processes incoming data, detects anomalies, and updates predictions in real-time. System retrains periodically for improved accuracy.

API Endpoints

Backend provides RESTful API for data management and predictions:

  • POST /api/battery-data - Add new battery measurements
  • GET /api/battery-history/{battery_id} - Get historical data
  • GET /api/predict-rul/{battery_id} - Get RUL prediction
  • POST /api/retrain-model - Retrain ML model
